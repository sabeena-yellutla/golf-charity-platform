# Golf Charity Platform

This is a frontend web application where users can submit golf scores, track activity, and view stats.

## Features
- Submit score functionality
- Recent activity tracking
- Wins calculation logic
- Data stored using localStorage
- Responsive design

## Tech Stack
- HTML
- CSS (Bootstrap)
- JavaScript

## Live Demo
https://golf-charity-site.vercel.app/

## Project Purpose
This project was built as part of an internship assignment to demonstrate frontend development skills, including layout design, responsiveness, and user experience.

## Note
This project currently focuses on frontend functionality. Backend and authentication are not implemented.
